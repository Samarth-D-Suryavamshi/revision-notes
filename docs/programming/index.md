# 💻 Programming & Languages

<div style="text-align: justify;"> Programming is the craft of transforming logic into executable software. This section covers the fundamental concepts that apply to all languages, as well as the unique quirks of popular industrial languages.</div>

## 📂 Topics Covered

### 1. [Programming Fundamentals](programming-fundamentals.md)
High-level concepts:

- **Memory Management**: Stack vs Heap, Garbage Collection.
- **Compilers vs Interpreters**: Bytecode, JIT, and native compilation.
- **Concurrency Models**: Threads, Event Loops, and Coroutines.

### 2. [Language Specifics](language-specific.md)
Deep dives into:

- **Python**: GIL, List comprehensions, Decorators, and AsyncIO.
- **JavaScript**: Prototype chain, Closures, Event Loop, and Promises.
- **C++/Java**: Pointers, Interfaces, Streams, and JVM internals.

---

## 🛠️ [The Developer's Mindset](the-developer-mindset.md)
Syntax is just the beginning. To be a great programmer, focus on:

- **Writeability**: How easy is it to write code?
- **Readability**: How easy is it for others to understand it?
- **Maintainability**: How easy is it to change without breaking things?
